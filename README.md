🌱 Recicle.bot 🌱

📌 Objetivo 📌
Tem o objetivo de ajudar os usuários a entenderem como descartar corretamente diferentes tipos de materiais, incentivando práticas sustentáveis e a redução do lixo doméstico, seja descartando ou reciclando o material para transformá-lo em decoração!✨

⚙️ O que faz? ⚙️
-	♻️ Identifica se um item é reciclável ou não ♻️
-  🗑️ Informa a forma correta de descarte 🗑️
-	📦 Classifica o material 📦
-	🎨 Manda ideais de reciclagem com vídeos 🎨
-	🎲 Gera vídeos aleatórios de artesanato quando alguém está no tédio 🎲

🤖 Comandos 🤖
!reciclagem
-Informe uma das opções de categorias
-Informe o material
-O bot dirá se o material é ou não reciclável, como descartar, sua categoria e um vídeo de artesanato para a reciclagem do objeto.
                   
!ajuda
-Aparecerá esta descrição de como o bot funciona.

!tedio
-Aparecerá um dos vídeos base de forma aleatória de artesanato com materiais recicláveis como recomendação para fazer no tempo livre.
🌱O bot vai te orientar sobre descarte e reutilização 🌱
